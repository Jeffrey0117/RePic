Features

Everything you need,
nothing you don't

RePic combines the speed of a native viewer with the power of web integration. No bloat, no subscriptions, no cloud dependency.

Blazing fast browsing

Virtual scrolling and preloaded thumbnails let you fly through thousands of images without a hiccup.

Web albums

Drag images from any website directly into RePic albums. Bypass hotlink protection automatically.

Virtual images (.repic)

Lightweight shortcuts to web images. A few bytes instead of megabytes, always up to date.

Drag to share

Drag any image out of RePic straight into Discord, LINE, social media, or any other app.

Non-destructive editing

Crop, annotate, and mosaic your images. All edits are saved separately — originals stay untouched.

Paste from clipboard

Press Ctrl+V to instantly import screenshots, copied images, or image URLs. Zero friction.

.repic format

Images without the weight

A .repic file is a tiny JSON shortcut that points to a web image. It stores the source URL and optional crop parameters — no pixel data. Double-click to open it instantly in RePic.
